# API Testing

A practical repository to learn REST API testing from scratch using Postman and other modern API testing tools.

The objective of this project is to document my learning process while practicing API testing as a QA Engineer. The repository will grow progressively with practical exercises, documentation, Postman collections and real-world testing scenarios.

---

# 🎯 Learning Goals

- Understand how REST APIs work
- Learn HTTP methods (GET, POST, PUT, PATCH, DELETE)
- Understand HTTP requests and responses
- Learn JSON structure
- Use Postman professionally
- Design API test cases
- Validate API responses
- Learn API authentication
- Create automated API tests
- Execute Postman collections with Newman
